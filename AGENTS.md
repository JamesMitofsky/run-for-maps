# AGENTS.md

Guidance for AI agents working in this repo.

## Human-only files — do not modify

AI agents must **never** create, edit, or delete:

- Any file under `docs/human/`.
- Any file whose first line (or top-of-file comment) contains `HUMAN ONLY`.

These are human-maintained. Read them for context, but leave them unchanged. If a
task seems to require editing one, stop and ask the human instead.
